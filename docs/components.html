Tabs

Use semantic tab roles, connect each tab with aria-controls, and add data-tabs for automatic behavior.

Protocol 2026-1842 is waiting for review.

Created on 06/13/2026 and assigned to Operations.

No files were attached.

HTML with automatic JavaScript behavior:

<div class="tabs" data-tabs id="request-tabs">
  <div class="tabs-list" role="tablist" aria-label="Request status">
    <button type="button" role="tab" id="request-summary-tab" aria-selected="true" aria-controls="request-summary">Summary</button>
    <button type="button" role="tab" id="request-history-tab" aria-selected="false" aria-controls="request-history">History</button>
    <button type="button" role="tab" id="request-files-tab" aria-selected="false" aria-controls="request-files">Files</button>
  </div>

  <div class="tabs-panel" role="tabpanel" id="request-summary" aria-labelledby="request-summary-tab">
    <p>Protocol 2026-1842 is waiting for review.</p>
  </div>
  <div class="tabs-panel" role="tabpanel" id="request-history" aria-labelledby="request-history-tab" hidden>
    <p>Created on 06/13/2026 and assigned to Operations.</p>
  </div>
  <div class="tabs-panel" role="tabpanel" id="request-files" aria-labelledby="request-files-tab" hidden>
    <p>No files were attached.</p>
  </div>
</div>

JavaScript API:

<script>
  LegacyCss.tabs.setup("#request-tabs");
  LegacyCss.tabs.select("#request-tabs", 1);
</script>

jQuery API:

<script>
  $("#request-tabs").tabs();
  $("#request-tabs").tabs("select", 1);
</script>
Modal dialog

The dialog below is opened by both the vanilla JavaScript API and the optional jQuery bridge.

Sign in

Use a semantic <dialog> element with .modal for the legacy chrome and close affordances.

HTML:

<p>
  <button type="button" onclick="LegacyCss.modal.open('#login-form')">Open with LegacyCss.modal.open()</button>
  <button type="button" onclick="$('#login-form').modal()">Open with $('#login-form').modal()</button>
</p>

<dialog id="login-form" class="modal" aria-labelledby="login-form-title">
  <header>
    <strong id="login-form-title">Sign in</strong>
    <button type="button" data-modal-close aria-label="Close dialog">Close</button>
  </header>
  <div>
    <p>Use a semantic <code>&lt;dialog&gt;</code> element with <code>.modal</code>.</p>
  </div>
  <footer>
    <button type="button" data-modal-close>Cancel</button>
    <button type="button" onclick="LegacyCss.modal.close('#login-form')">Close via API</button>
  </footer>
</dialog>

JavaScript API:

<script>
  LegacyCss.modal.open("#login-form");
  LegacyCss.modal.close("#login-form");
  LegacyCss.modal.toggle("#login-form");
</script>

jQuery API:

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
<script>
  $("#login-form").modal();
  $("#login-form").modal("close");
  $("#login-form").modal("toggle");
</script>
